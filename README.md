# 开发规范手册

## 1 原则

- 坚持遵循同一套代码规范，代码同出一门，保持整洁、易读、一致，更容易被理解和被维护。
- 对待规范，要严格遵守；对待风格，要懂得尊重。
- 如果你想要为这个规范做贡献或觉得有不合理的地方，请访问New Issue。
